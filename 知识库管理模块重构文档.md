### 1.重构范围

覆盖**三层级页面**：

1. 知识库管理页（总览 + 知识库列表）
2. 文档管理页（指定知识库下的文档列表）
3. 分块管理页（指定文档下的分块列表）

### 2.需求分析

#### 2.1 核心功能需求

2.1.1 知识库管理页

| 功能模块   | 具体需求                                                     |
| ---------- | ------------------------------------------------------------ |
| 统计概览   | 展示 3 项核心统计数据：1. 知识库总数2. 全部文档数3. 含文档的知识库数。每项统计卡片右侧提供「全部」筛选入口 |
| 知识库列表 | 表格展示，列包括：名称（可点击跳转至文档管理页）、Embedding 模型、Collection（标签样式）、文档数、负责人、创建时间、修改时间、操作（编辑 / 删除按钮） |
| 操作栏     | 顶部提供：搜索框（按知识库名称搜索）、搜索按钮、刷新按钮、「新建知识库」按钮 |
| 分页       | 底部展示数据总数、分页导航（上一页 / 下一页 / 页码）         |

2.1.2 文档管理页

| 功能模块 | 具体需求                                                     |
| -------- | ------------------------------------------------------------ |
| 页面标识 | 顶部面包屑：`知识库管理 / 文档管理`，下方显示当前知识库名称及 Collection 标识 |
| 文档列表 | 表格展示，列包括：文档名称（可点击跳转至分块管理页）、来源、处理模式、状态（绿色`success`标识）、启用开关、分块数、类型、大小、更新时间、操作（编辑 / 查看分块 / 下载 / 删除按钮） |
| 操作栏   | 顶部提供：返回知识库按钮、「上传文档」按钮；列表上方提供：搜索框（按文档名称搜索）、搜索按钮、状态筛选下拉框、刷新按钮 |
| 分页     | 底部展示数据总数、分页导航                                   |

2.1.3 分块管理页

| 功能模块 | 具体需求                                                     |
| -------- | ------------------------------------------------------------ |
| 页面标识 | 顶部面包屑：`知识库管理 / 文档管理 / 切片管理`，下方显示当前文档名称及知识库 ID |
| 分块列表 | 表格展示，列包括：复选框（用于批量操作）、序号、内容（预览截断）、状态（启用 / 禁用标签）、字符数、Token 数、更新时间、操作（编辑 / 禁用 / 删除按钮） |
| 操作栏   | 顶部提供：返回文档按钮、重建向量按钮、「新建分块」按钮；列表上方提供：状态筛选下拉框、刷新按钮、批量操作按钮组（批量启用 / 批量禁用 / 全量启用 / 全量禁用） |
| 分页     | 底部展示数据总数、分页导航                                   |

### 3.模块拆分

| 模块       | 核心职责                                                     |
| ---------- | ------------------------------------------------------------ |
| 知识库模块 | 管理知识库的创建、编辑、删除、列表查询、统计数据计算         |
| 文档模块   | 管理文档的上传、编辑、删除、状态切换、列表查询，关联所属知识库 |
| 分块模块   | 管理分块的创建、编辑、删除、状态切换、批量操作，关联所属文档，支持向量重建 |

## 4.UI/UX 设计规范

### 4.1 布局规范

- 全局布局：

  - 顶部：面包屑导航 + 操作按钮组
  - 中部：主内容区（统计卡片 / 表格）
  - 底部：分页组件

  

- 颜色规范：

  - 状态色：绿色（`#00B42A`）用于`success`状态

  

- 组件规范：

  - 统计卡片：圆角
  - 表格：浅灰边框、行 hover 高亮、表头加粗
  - 按钮：圆角
  - 标签：圆角、浅背景色，用于 Collection、状态展示
  - 开关：蓝色激活态、灰色禁用态

  

### 4.2 页面交互细节

1. 知识库列表：点击名称跳转至对应文档管理页
2. 文档列表：点击名称跳转至对应分块管理页，开关切换实时更新状态
3. 分块列表：勾选复选框后可执行批量操作，内容预览最多显示 2 行
4. 操作反馈：所有增删改操作后弹出 Toast 提示成功 / 失败
